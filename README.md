# How to add on IPTV Player ?

1- Download a IPTV Player.

2- Click on "Create Playlist".

3- Click on "New M3U Playlist" Bouton.

4- Copy, Paste the link of the iptv Player.

5- Add a Name.

6- Click on the OK bouton and enjoy.
